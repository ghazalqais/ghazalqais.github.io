MedClassify - AI Medical Assistant

A production-ready Arabic medical AI web application that analyzes patient symptoms and recommends the appropriate medical specialty. Using a fine-tuned MARBERT model trained on 25,000+ annotated medical consultations, MedClassify classifies symptoms into 5 specialties: Dentistry, Ophthalmology, Orthopedics, Psychiatry, and Nutrition.

Methodology

The approach combines:

  • Systematic comparison of 20+ ML approaches across traditional ML, neural networks, and transformers
  • MARBERT fine-tuning with 4 epochs, learning rate 2e-5, batch size 16
  • Custom Arabic preprocessing pipeline handling tashkeel removal and normalization
  • Stratified 70/15/15 train/validation/test split maintaining class balance
  • Enhanced preprocessing with ISRI stemming and stop word removal

Key Results

  • MARBERT: 90%+ test accuracy (selected model for deployment)
  • Traditional ML: 85-89% accuracy with Logistic Regression + TF-IDF
  • Neural Networks: 83-88% accuracy with GRU/LSTM + embeddings
  • Enhanced preprocessing improved accuracy by 3-7% across all models
  • Real-time inference with bilingual Arabic-English web interface

Production Deployment

  • Responsive web application with native Arabic RTL support
  • Flask API backend with PyTorch inference pipeline
  • Smart examples for quick testing across all 5 medical specialties
  • Privacy-focused design with no data storage
  • Deployed on Heroku with Docker containerization

Brain Tumor MRI Classification

A comprehensive deep learning system for automated brain tumor classification from MRI scans. This project implements and compares four state-of-the-art CNN architectures to classify brain tumors into glioma, meningioma, pituitary tumors, and normal tissue with clinical-grade accuracy.

Methodology

The approach combines:

  • Systematic comparison of four CNN architectures: EfficientNet-B3, DenseNet-121, ResNet-50, and VGG-16
  • 2×2 factorial hyperparameter optimization design with learning rate and dropout tuning
  • Comprehensive dataset of 7,023 professionally annotated MRI scans
  • Stratified train/validation/test split (70%/15%/15%) maintaining class balance
  • Transfer learning with ImageNet pre-trained weights and medical image preprocessing

Key Results

  • EfficientNet-B3: 99.24% test accuracy (best performer) with optimal efficiency trade-off
  • DenseNet-121: 97.82% test accuracy with excellent feature reuse capabilities
  • VGG-16: 90.70% test accuracy demonstrating classical architecture effectiveness
  • ResNet-50: 87.19% test accuracy with architectural limitations identified
  • Dataset size analysis showing 96.8% accuracy achievable with just 30% of training data
  • Real-time inference: 70.6ms CPU, 14.2ms GPU per image with EfficientNet-B3

Clinical Impact

  • Web-based interface for real-time brain tumor classification
  • Support for four tumor types: glioma, meningioma, pituitary, and normal tissue
  • Multi-model comparison capability for enhanced diagnostic confidence
  • Production deployment with Flask framework and containerized architecture

Image Deep Fake Detection

This project addresses the growing concern of deepfake images by developing a robust detection system using convolutional neural networks and transfer learning. The model achieves 97.40% accuracy by analyzing patterns in real and AI-generated facial images.

Methodology

The approach combines:

  • Utilization of three CNN architectures: DenseNet121, ResNet50, and InceptionV3
  • Transfer learning to improve performance with limited training data
  • Balanced dataset of real images (FFHQ) and fake images (StyleGAN)
  • Comprehensive evaluation using precision, recall, and F1-score metrics

Key Results

  • DenseNet121: 97.40% accuracy with 0.067 loss (best performer)
  • InceptionV3: 97.23% accuracy with 0.073 loss
  • ResNet50: 85.67% accuracy with 0.438 loss
  • DenseNet121 achieved balanced precision (0.99 for fake, 0.96 for real) and recall (0.96 for fake, 0.99 for real)
